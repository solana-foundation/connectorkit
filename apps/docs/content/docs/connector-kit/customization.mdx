## Quick Customization Guide

### Pre-built Themes

Choose from 4 Solana-optimized themes:

```tsx
import { ConnectButton, solanaTheme, phantomTheme, minimalTheme, darkTheme } from '@connector-kit/connector'

function App() {
  return <ConnectButton theme={solanaTheme} />
}
```

### Custom Themes

Create branded themes with partial overrides:

```tsx
import { mergeThemeOverrides, solanaTheme } from '@connector-kit/connector'

const brandTheme = mergeThemeOverrides(solanaTheme, {
  colors: { primary: '#FF6B6B', accent: '#4ECDC4' },
  button: { height: 56, shadow: 'xl' },
})

<ConnectButton theme={brandTheme} />
```

### Simplified Configuration

Use `getDefaultConfig` for quick setup:

```tsx
import { AppProvider, getDefaultConfig } from '@connector-kit/connector'

const config = getDefaultConfig({
  appName: 'My App',
  network: 'mainnet-beta',
  autoConnect: true,
})

<AppProvider connectorConfig={config}>{children}</AppProvider>
```

### Modal Control

Programmatically manage modals:

```tsx
import { useModal, modalRoutes } from '@connector-kit/connector'

function WalletControls() {
  const modal = useModal()
  
  return (
    <div>
      <button onClick={modal.openWallets}>Connect</button>
      <button onClick={modal.openProfile}>Profile</button>
      <button onClick={() => modal.open(modalRoutes.ACCOUNT_SETTINGS)}>Settings</button>
    </div>
  )
}
```

### Wallet Filtering

Filter wallets by capabilities:

```tsx
import { getPopularWallets, getMobileWallets } from '@connector-kit/connector'

const popularWallets = getPopularWallets() // Phantom, Solflare, etc.
const mobileWallets = getMobileWallets()   // Mobile-first wallets
```

### Custom Connect Button

Build your own connect experience:

```tsx
import { useConnector, useModal } from '@connector-kit/connector'

function CustomButton() {
  const { connected, connecting, selectedWallet } = useConnector()
  const modal = useModal()
  
  if (connected) {
    return <span>{selectedWallet?.name} Connected</span>
  }
  
  return (
    <button onClick={modal.openWallets} disabled={connecting}>
      {connecting ? 'Connecting...' : 'Connect Wallet'}
    </button>
  )
}
```

### Environment Setup

Configure per environment:

```tsx
const config = getDefaultConfig({
  appName: 'My App',
  network: process.env.NODE_ENV === 'production' ? 'mainnet-beta' : 'devnet',
  debug: process.env.NODE_ENV === 'development',
})
```

## 3. Headless Implementation

Complete control over UI and behavior:

```tsx
import { useConnector } from '@connectorkit/connector-kit'
import { useWalletAddress, useBalance } from '@connectorkit/hooks'

function CustomWalletConnect() {
  const { wallets, select, disconnect } = useConnector()
  const { address, connected } = useWalletAddress()
  const { balance } = useBalance({ address })

  return (
    <div>
      {!connected ? (
        wallets?.map(wallet => (
          <button 
            key={wallet.name}
            onClick={() => select(wallet.name)}
            disabled={!wallet.installed}
          >
            Connect {wallet.name}
          </button>
        ))
      ) : (
        <div>
          <p>Connected: {address}</p>
          <p>Balance: {Number(balance) / 1e9} SOL</p>
          <button onClick={disconnect}>Disconnect</button>
        </div>
      )}
    </div>
  )
}
```

## 4. Advanced Features

Access wallet capabilities and sign transactions:

```tsx
function TransactionSigner() {
  const { selectedWallet } = useConnector()
  
  const handleSign = async () => {
    const features = selectedWallet?.features
    const signFeature = features?.['solana:signTransaction']
    
    if (signFeature) {
      // Your transaction signing logic
    }
  }
  
  return <button onClick={handleSign}>Sign Transaction</button>
}
```

## Choose Your Approach

- **Pre-built Components** - Fastest setup, minimal customization
- **Interactive Demo** - Rich UX with animations and wallet management
- **Headless Implementation** - Maximum flexibility and control
- **Advanced Features** - Transaction signing and wallet capabilities

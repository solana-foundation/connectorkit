# API Reference

Complete reference for all ConnectorKit APIs, components, and utilities.

## Components

### ConnectButton

The main wallet connection component with built-in modal and account management.

```tsx
import { ConnectButton } from '@connector-kit/connector'

<ConnectButton 
  size="md"                    // "sm" | "md" | "lg"
  variant="default"            // "default" | "icon-only" | "minimal" | "outline"
  label="Connect Wallet"       // Custom button text
  disabled={false}             // Disable button
  loading={false}              // Show loading state
  className="custom-styles"    // Custom CSS classes
/>
```

**Props:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Button size |
| `variant` | `"default" \| "icon-only" \| "minimal" \| "outline"` | `"default"` | Button style variant |
| `label` | `string` | `"Connect Wallet"` | Custom button text |
| `disabled` | `boolean` | `false` | Disable the button |
| `loading` | `boolean` | `false` | Show loading spinner |
| `className` | `string` | - | Custom CSS classes |
| `theme` | `ConnectorTheme` | - | Custom theme overrides |

### ConnectorProvider

Root provider that manages wallet connections and configuration.

```tsx
import { ConnectorProvider, getDefaultConfig } from '@connector-kit/connector'

<ConnectorProvider 
  config={getDefaultConfig({ appName: "Your App" })}
  mobile={getDefaultMobileConfig({ appName: "Your App" })}
>
  {children}
</ConnectorProvider>
```

**Props:**

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `config` | `ConnectorConfig` | ✓ | Main configuration object |
| `mobile` | `MobileWalletAdapterConfig` | ✗ | Mobile wallet adapter config |

### ConnectModal

Standalone modal for wallet connection (automatically included with ConnectButton).

```tsx
import { ConnectModal } from '@connector-kit/connector'

<ConnectModal options={{ autoCloseOnConnect: true }} />
```

## Hooks

### useConnector

Main hook for wallet connection state and methods.

```tsx
import { useConnector } from '@connector-kit/connector'

function WalletStatus() {
  const {
    // State
    connected,
    connecting,
    wallets,
    selectedWallet,
    accounts,
    selectedAccount,
    modalOpen,
    modalRoute,
    
    // Methods
    select,
    disconnect,
    selectAccount,
    openModal,
    closeModal
  } = useConnector()
}
```

**Returns:**

| Property | Type | Description |
|----------|------|-------------|
| `connected` | `boolean` | Whether a wallet is connected |
| `connecting` | `boolean` | Whether connection is in progress |
| `wallets` | `WalletInfo[]` | Available wallets |
| `selectedWallet` | `Wallet \| null` | Currently connected wallet |
| `accounts` | `AccountInfo[]` | Available accounts |
| `selectedAccount` | `string \| null` | Selected account address |
| `modalOpen` | `boolean` | Modal visibility state |
| `modalRoute` | `ModalRoute` | Current modal route |
| `select(name)` | `(name: string) => Promise<void>` | Connect to wallet by name |
| `disconnect()` | `() => Promise<void>` | Disconnect current wallet |
| `selectAccount(address)` | `(address: string) => Promise<void>` | Switch account |
| `openModal(route?)` | `(route?: ModalRoute) => void` | Open modal |
| `closeModal()` | `() => void` | Close modal |

### useModal

Hook for managing modal state and navigation.

```tsx
import { useModal } from '@connector-kit/connector'

function ModalControls() {
  const {
    isOpen,
    route,
    open,
    close,
    openWallets,
    openProfile,
    openSettings
  } = useModal()
}
```

**Returns:**

| Property | Type | Description |
|----------|------|-------------|
| `isOpen` | `boolean` | Whether modal is open |
| `route` | `ModalRoute` | Current modal route |
| `open(route?)` | `(route?: ModalRoute) => void` | Open modal with optional route |
| `close()` | `() => void` | Close modal |
| `openWallets()` | `() => void` | Open wallet selection |
| `openProfile()` | `() => void` | Open user profile |
| `openSettings()` | `() => void` | Open settings |

## Utility Functions

### getDefaultConfig

Creates a default connector configuration with sensible defaults.

```tsx
import { getDefaultConfig } from '@connector-kit/connector'

const config = getDefaultConfig({
  appName: 'My App',              // Required: App name for wallet prompts
  appUrl: 'https://myapp.com',    // Optional: App URL
  autoConnect: true,              // Optional: Auto-reconnect (default: true)
  debug: false,                   // Optional: Debug logging (default: dev mode)
  network: 'mainnet-beta',        // Optional: Solana network (default: mainnet-beta)
  enableMobile: true,             // Optional: Enable mobile support (default: true)
})
```

### getDefaultMobileConfig

Creates default mobile wallet adapter configuration.

```tsx
import { getDefaultMobileConfig } from '@connector-kit/connector'

const mobileConfig = getDefaultMobileConfig({
  appName: 'My App',
  appUrl: 'https://myapp.com',
  network: 'mainnet-beta',
})
```

## SDK Integration

ConnectorKit integrates with `@connector-kit/sdk` for Solana operations:

```tsx
import { useWalletAddress, useBalance, useTransaction } from '@connector-kit/sdk'
import { useConnector } from '@connector-kit/connector'

function WalletIntegration() {
  const { connected } = useConnector()
  const { address } = useWalletAddress()
  const { balance } = useBalance({ address })
  const { sendTransaction } = useTransaction()
  
  const handleSend = async () => {
    try {
      const signature = await sendTransaction({
        to: 'recipient-address',
        amount: 1000000 // 0.001 SOL in lamports
      })
      console.log('Sent:', signature)
    } catch (error) {
      console.error('Failed:', error)
    }
  }
  
  return (
    <div>
      {connected ? (
        <div>
          <p>Address: {address}</p>
          <p>Balance: {balance ? Number(balance) / 1e9 : 0} SOL</p>
          <button onClick={handleSend}>Send SOL</button>
        </div>
      ) : (
        <p>Connect wallet to continue</p>
      )}
    </div>
  )
}
```
